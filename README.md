# 海克斯大乱斗 AI 辅助

本地网页工具：输入英雄名，自动解析英雄，抓取最新攻略源并汇总海克斯强化、出装路线、黑科技判断和来源链接。

## 启动

```bash
npm start
```

然后打开：

```text
http://localhost:4173
```

## 设计取舍

- 英雄数据优先使用 Riot Data Dragon 最新版本，断网时保留少量常用英雄兜底。
- 攻略源优先使用会随版本更新的统计站：Mobalytics、U.GG、Lolalytics、OP.GG、Arena Assistant。
- 抖音和小红书反爬较强，工具提供一键检索入口，不把单条视频当成主结论。
- 没有抓到结构化数据时，会输出按英雄机制推导的保守方案，避免硬编不存在的“最新答案”。
